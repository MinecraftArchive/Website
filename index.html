/// The Idea
    People used Google and Websites liked 4shared.com and chomikuj.pl, but it doesnt always work.Then Minecraft Modding Archive was born. We hope to help you all and would like community suggestions. The Minecraft Modding Archive project seeks to preserve and save every mod in existance. If we didn’t do this stuff like Technic 2 may be lost in existance. Thats why were here to help you save the day. If we wern’t here, No one could ever find them again.
/// Why?
    Most mod authors stop with their download links, after they think its not needed anymore. When these authors eventually decide to shut down these links, the mod is then rendered unplayable and unable to be enjoyed by future generations. By archiving these mods, along with additional community generated content. These games will be available to play long after the mod developers have abandoned them!
/// The Features
  1. We hope to archive every single version of Forge, Modloader, ModloaderMP, Cauldron Craftbukkit, Spigot, etc.
  2. Collect guides for each version of mods, including installation and use.
  3. Find Mod Spotlights for each mod.</li>
  4. Find out who is the author of each mod is.
  5. Keep extensive documentation on mod creation, mod porting (MCPortcentral), plugin creation, and instructions for how to compile each mod/plugin/tool. With all the required tools for creation archived as well (Including MCP, and all mod creation dependencies).
/// How?
  1. Old/Discontinued mods will be obtained by contacting the original developer, searching through old modpack archives, and by community-sourcing via IRC/Reddit.
  2. Recent mods will be added manually as well, and by scraping existing mod hosting services
  3. Contributors will create curated pages and community forums for each mod.
  4. Also welcome to join our forum service and keep in contact with us.
/// Getting Envolved!
  1. If you’d like to assist in the archival process, please join us on our Discord
  2. We’d really appreciate help finding legitimate files for mods and plugins that are no longer available due to dead links, thread closings, etc. If you find any such files, we’d love to hear about it on the Discord channel.
/// Applications!
  1. Web Developers - Create a simple, informative web interface.
  2. Community Managers - Curate the pages for mods, plugins, and server tools, moderate on the Discord channel.
  3. Documentation Specialists - Assemble/create helpful guides on installation and use of the mods, plugins, and server administration tools. This position demands an in-depth knowledge of modding early in the history of the game.
  4. Moderators - Simply help moderate the discord.
    If you think you can help in some other meaningful way, talk to us on our Discord chat, we’re happy to take all the help we can get!
